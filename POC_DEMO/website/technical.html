Technical Overview

Fast, clear, and reliable guidance.

ML Stack

  • Scene & Hazards: Google Gemini (2.5 Flash / 2.0 Flash) chosen by context for speed vs. depth.
  • Navigation Guidance: Maps instruction + image → normalized schema + haptic recommendation.
  • TTS: Coqui VITS tuned for clarity and low listener fatigue.
  • Transcription: Whisper (Base) for spoken input.

Key Capabilities

  • Hazard detection with bearing, proximity, confidence, and haptics.
  • Scene descriptions that prioritize orientation and safety.
  • Traffic signal understanding with Deep Analyze for crossings.

TTS Results

Quality, latency, RTF, and WER comparisons used to select voices.
